## 设置

- `F1` 输入： `screencast`，切换截屏模式，方便录屏

## 快捷键

| 快捷键                           | 说明             |
| -------------------------------- | ---------------- |
| `ctrl` + `c`                     | 复制光标当前行   |
| `ctrl` + `v`                     | 粘贴到光标当前行 |
| `ctrl` + `d`                     | 选中字符         |
| `ctrl` + `shift` + `alt` + `↑/↓` | 多行选中         |
| `ctrl` + `k`, `ctrl` + `0`       | 折叠所有         |
